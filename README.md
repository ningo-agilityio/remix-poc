# Remix Poc
This is a basic sample for Remix
## Features:
- Blogs list
- Create new blog

### How to run
- Clone repo:
```
git clone git@github.com:ningo-agilityio/remix-poc.git
```

- Install dependencies:
```
yarn install
```

- Run app in local
```
yarn dev
```

- View app via localhost:3000
